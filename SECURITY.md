# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

You can report a vulnerability by [opening an issue](https://github.com/open-data-plan/g2plot-react/issues) on this repository.
